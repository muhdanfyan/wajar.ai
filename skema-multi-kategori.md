# Skema Pembayaran Multi-Kategori & Multi-Paket

> Bagaimana kalau user mau belajar lebih dari satu kategori?
> Apakah bayarnya sama atau dibedakan?

---

## 🧠 Prinsip Dasar

**WAJAR.AI = 1 akun, akses ke SEMUA kategori.**

User bayar berdasarkan **waktu pemakaian**, bukan berdasarkan kategori.
Kenapa? Karena Aiman ini satu — ngajar Matematika punya, ngajar Coding juga bisa, curhat juga bisa.

**Tidak ada harga per kategori.** Cukup pilih paket berdasarkan durasi.

---

## 💳 Tabel Harga (Tetap — Tidak Berubah)

| Paket | Durasi | Harga | Cocok untuk |
|:-----:|:------:|:-----:|:-----------:|
| 🎁 Gratis | 8.000 kata | Rp0 | Coba-coba, satu sesi tanya jawab |
| ☕ Per Jam | ~1 jam | Rp1.000 | Dadakan, satu topik cepat |
| 📅 Per Hari | ~24 jam | Rp10.000 | Belajar intensif satu hari penuh |
| 📆 Per Minggu | 7 hari | Rp50.000 | Project mingguan, ujian, persiapan |
| 🌙 Per Bulan | 30 hari | Rp150.000 | Paling hemat untuk pengguna tetap |

## 📋 Skenario Multi-Kategori

### Skenario 1: User Belajar 2+ Kategori
```
User: "Aku mau belajar Matematika sama Coding"
Aiman: "Bisa! Sekarang mau mulai dari Matematika dulu atau Coding?"
       "Per Jam Rp1.000 — bisa tanya apa aja, campur-campur."
```
✅ **Bayar sekali, akses semua kategori.** Gak perlu bayar terpisah.

### Skenario 2: User + Keluarga
```
User: "Bisa dipake berdua sama adik?"
Aiman: "Bisa! Satu akun bisa dipake berdua. Tapi gantian ya chatnya."
```
✅ **1 akun = 2 user (maks).** Harga tetap sama.

### Skenario 3: User Belajar + Curhat
```
User: "Belajar Coding dulu, nanti curhat boleh?"
Aiman: "Tentu! Di paket yang sama boleh tanya apa aja.
        Mau Coding dulu atau cerita dulu?"
```
✅ **Campur topik — satu harga.** Gak perlu bayar tambahan.

## 🎯 Rekomendasi untuk User

| Kebutuhan User | Rekomendasi Aiman |
|:---------------|:-----------------:|
| Mau coba dulu | 🎁 Gratis 8.000 kata |
| Tanya 2-3 soal | ☕ Rp1.000 (Per Jam) |
| Project coding + belajar + curhat | 📅 Rp10.000 (Per Hari) |
| Ujian seminggu + mental support | 📆 Rp50.000 (Per Minggu) |
| Belajar rutin tiap hari | 🌙 Rp150.000 (Per Bulan) |

---

## ⚠️ Catatan Penting

1. **Tidak ada paket bundling spesial** — semua kategori satu harga
2. **Tidak ada diskon multi-user** — max 2 orang per akun
3. **Tidak ada uang muka / deposit** — bayar sesuai pemakaian
4. **Tidak ada paket tahunan** — cukup bulanan
5. **GRATIS 8.000 kata** — user bisa cobain SEMUA kategori tanpa bayar
